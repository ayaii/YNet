Wired Vs Wireless

Wired Network

A Wired Internet Connection refers to the transmission of data with the use of ethernet cables. Wired Internet has many positives such as faster internet speeds, reliability and cheaper costs when compared to a wireless connection although using a wired connection has many positives it still has many negatives such as the lack of mobility and having to use ethernet cables when wanting to connect another device when wanting to connect to the internet.

Wireless Network

A Wireless Internet Connection is a network that is not connected with the use of cables. Advantages of a wireless network include mobility and freedom while disadvantages include slower internet speeds, less stable and connection loss based on multiple factors such as distance and walls causing signal to be blocked or lossed.
