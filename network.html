Building a Wireless Network

Depending on your current internet setup you are able to start a wireless network either using a Wireless Access Point also known as WAP or a Wireless Router. A Wireless Access Point is a device that receives data via an ethernet cable which then converts this data into either a 2.4ghz or 5ghz wireless signal while a Wireless Router is a device that performs the same functions of a router but it also includes the functions of a wireless access point.
